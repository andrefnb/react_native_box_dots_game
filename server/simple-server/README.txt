Simple Server
-------------

Simple Server provides a small common API on top of AllegroServe
and Hunchentoot. It is documented at

http://www.cs.northwestern.edu/academics/courses/325/readings/web-services.php#simple-server

JSON Demo
---------

Is a small example web app using Simple Server to demonstrate 
accessing static HTML assets and JSON communication between
JavaScript on the client and Lisp on the server. It is documented at

http://www.cs.northwestern.edu/academics/courses/325/readings/web-services.php#json-demo
